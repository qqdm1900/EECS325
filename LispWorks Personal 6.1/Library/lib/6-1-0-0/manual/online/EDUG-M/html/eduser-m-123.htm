4.11.2 Removing multiple definitions

Undefine Buffer

Editor Command

Argument: None
Key sequence: None

Undefines all the definitions in the current buffer.

Undefine Region

Editor Command

Argument: None
Key sequence: None

Undefines the definitions in the current region.
