1.4.5 Licensing issues

Runtime files that are created using Delivery with LispWorks for Windows, LispWorks for Linux, LispWorks for FreeBSD, LispWorks for x86/x64 Solaris and LispWorks for Macintosh Professional and Enterprise Editions do not require a runtime license.

Runtime files generated by LispWorks (32-bit) for UNIX do require a LispWorks runtime license. See Runtime licensing on UNIX for more information.
