:editor-commands-to-delete

Keyword

Default value: :all-groups

When the Editor is loaded, you can delete some of its commands by passing a list of them with this keyword. Note that, by default, most Editor commands are retained. See Editors for delivered applications for more details.

Affected by: :keep-debug-mode.
