:size-t

FLI type descriptor
Summary

Converts between a Lisp integer and an ISO C size_t.

Syntax

:size-t

Arguments

None.

Description

The FLI :size-t type converts between a Lisp integer and an ISO C size_t type, which is an unsigned integer representing the size of an object in bytes.

See also

:ssize-t
