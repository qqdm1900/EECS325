:fixnum

FLI type descriptor
Summary

Converts between a Lisp fixnum and a 32 bit raw integer.

Syntax

:fixnum

Arguments

None.

Description

The FLI :fixnum type converts between a Lisp fixnum and a 32 bit integer in C.
