*null-pointer*

Variable
Summary

A null pointer.

Package

fli

Description

The variable *null-pointer* contains a (:pointer :void) with address 0.

This provides a simple way to pass a null pointer when needed.

Example
(fli:pointer-address fli:*null-pointer*)
=>
0
 
(fli:null-pointer-p fli:*null-pointer*)
=>
T
See also
