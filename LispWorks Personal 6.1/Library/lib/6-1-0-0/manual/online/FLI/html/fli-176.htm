:lisp-float

FLI type descriptor
Summary

Converts between any Lisp float and the C double type or the C float type.

Syntax

:lisp-float &optional float-type

float-type ::= :single | :double

Arguments

float-type

Determines the C type to convert to. The default is :single .

Description

The FLI :lisp-float type converts between any Lisp float and either the C float or the C double type. The default is to convert to the C float type, but by specifying :double for float-type , conversion occurs between any Lisp float and the C double type.
