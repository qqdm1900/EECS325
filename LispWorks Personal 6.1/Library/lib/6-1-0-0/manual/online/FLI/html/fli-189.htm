:signed

FLI type descriptor
Summary

Converts between a Lisp integer and a foreign signed integer.

Syntax

:signed &optional integer-type 

integer-type
 ::= :byte | :char | :short | :int |
              :long | :long :int | :short :int
Arguments

integer-type

The type of the signed integer.

Description

The :signed type converts between a Lisp integer and a foreign signed integer. The optional integer-type argument specifies other kinds of signed integer types. See Table A comparison of Lisp and C signed types for a comparison between Lisp and C signed types.
