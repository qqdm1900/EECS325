:float

FLI type descriptor
Summary

Converts a Lisp single float to a C float.

Syntax

:float

Arguments

None.

Description

The FLI :float type converts between a Lisp single float and the C float type.

Compatibility note

In LispWorks 4.4 and previous on Windows and Linux platforms, all Lisp floats are doubles. In LispWorks 6.1, there are three disjoint Lisp float types, on all platforms.
