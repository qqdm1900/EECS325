:ptrdiff-t

FLI type descriptor
Summary

Converts between a Lisp integer and an ISO C ptrdiff_t.

Syntax

:ptrdiff-t

Arguments

None.

Description

The FLI :ptrdiff-t type converts between a Lisp integer and an ISO C ptrdiff_t type, which is an signed integer representing the difference in bytes between two pointers.
