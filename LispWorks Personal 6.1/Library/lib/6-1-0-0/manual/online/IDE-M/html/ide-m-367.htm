17.5.5 Copying in the Inspector

You can easily copy objects in the inspector, ready for pasting into other tools.

To copy the inspected object itself use Object > Copy

To copy a slot value use Slots > Copy .

To copy an attribute use Slots > Attributes > Copy .

Similarly you can use Object > ClipSlots > Clip or Slots > Attributes > Clip to place the object itself, a slot value or an attribute on the Object Clipboard, so that you can conveniently retrieve them later. See The Object Clipboard for details.
