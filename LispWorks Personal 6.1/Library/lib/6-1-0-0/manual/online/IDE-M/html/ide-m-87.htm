3.16.1 'Tabbing' between buttons

On Mac OS X, by default you cannot use the Tab key to move the focus from one dialog button to another. You can fix this by selecting Apple Menu > System Preferences... Keyboard & Mouse > Keyboard Shortcuts > Full keyboard access: > All controls

This setting makes Tab switch between the buttons and also Space will select the Yes button in a dialog by default.
