7.1 LispWorks menus

When the LispWorks IDE starts, the standard application menu LispWorks becomes available.

Also available are the File and Window menus. Other menus may also be available depending on the current tool. By default, the LispWorks IDE starts a Listener tool so you will see the menu bar for that tool. But, whatever LispWorks tool is current,. you will always have the LispWorks, File and Window menus.
