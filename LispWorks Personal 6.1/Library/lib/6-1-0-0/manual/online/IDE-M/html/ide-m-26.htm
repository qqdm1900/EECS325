3.1.4 Displaying tools using the keyboard

Accelerators are provided for the popular items on the Tools menu.

Each tool accelerator is an alphanumeric key together with platform-specific modifier keys as shown in Tool accelerator keys.

You cannot configure these pre-defined tool accelerators.
