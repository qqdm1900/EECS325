25.6.5 File description area

The Description: area displays information about any system member selected in the Files area. If no such member is selected, information about the current system (the one named in the System area) is shown instead. The same pieces of information are shown as in the tree view. See Description area for details. As with other views, items selected in this area can be operated on using commands in the Description menu.
