15.2.2 Signatures button

Click Signatures... to display the Method Signatures dialog shown in Method Signatures dialog. This dialog lists the signatures for the methods defined on the current generic function. The signature of a method shows the types of the arguments.

Figure 15.3 Method Signatures dialog

To list the method combinations of any defined method in the Generic Function Browser, select its signature from the list in the Signatures panel of the Method Signatures dialog and click OK .

You can restrict the signatures displayed using Filter box in the usual way.

You can also restrict the display with the Restricted Class box. See Restricting displayed signatures by class for details.
