26.2.1 Sorting entries

Entries in the Window Browser can be sorted using the Sort panel in the General tab in the Preferences dialog. Choose the sort option you require from the list available.

By Name

Sorts items alphabetically by name.

By Package

Sorts items alphabetically by package name.

Unsorted

Displays items in the order they are defined in. This is the default setting.
