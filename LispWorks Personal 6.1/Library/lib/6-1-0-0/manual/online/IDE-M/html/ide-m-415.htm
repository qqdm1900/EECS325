22.2.2 Cumulative Results

Figure 22.3 The Profiler's Cumulative Results view

  • The main list displays the functions called during the last profile. For each function, the list displays the number of times it was found on the stack and the number of times it was found at the top of the stack.
  • The Filter box lets you restrict the display of information in the Results area.
