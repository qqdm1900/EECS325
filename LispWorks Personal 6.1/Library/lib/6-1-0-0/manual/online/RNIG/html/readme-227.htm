11.4.1.6 SPARC Solaris (LispWorks 64-bit)

The files you need to unpack for LispWorks (64-bit) on Solaris are lw61-sparc64.tar and lwdoc61-sparc64.tar .

The LispWorks image is

./lispworks-6-1-0-sparc64-solaris
