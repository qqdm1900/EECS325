11.4.1.5 SPARC Solaris (LispWorks 32-bit)

The files you need to unpack for LispWorks (32-bit) on Solaris are lw61-sparc.tar and lwdoc61-unix.tar .

The LispWorks image is

./lispworks-6-1-0-sparc-solaris
