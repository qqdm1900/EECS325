5.3.4 Selecting the correct software package file

The LispWorks (32-bit) Professional/Enterprise software package file is called LispWorks 61 -32bit and can be found in the x86-solaris directory of the LispWorks 6.1 CD-ROM. The package name is LispWorks61-32bit .

The LispWorks (64-bit) Enterprise software package file is called LispWorks 61 -64bit and can be found in the amd64-solaris directory of the LispWorks 6.1 CD-ROM. The package name is LispWorks61-64bit .

The Personal Edition software package file is called LispWorksPersonal 61 -32bit. The package name is LispWorksPersonal61-32bit .

Note: the software may be supplied in a compressed format with a .gz extension. Uncompress it using gunzip .
