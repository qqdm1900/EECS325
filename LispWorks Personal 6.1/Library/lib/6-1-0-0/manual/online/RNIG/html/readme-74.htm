4.4.3.4 Installing CLIM 2.0

The following module is packaged as a separate RPM file for installation after the main lispworks package. It is available in all LispWorks Editions except the Personal Edition.

File distributions for layered products in Professional and Enterprise Editions

File Distribution

Layered Product

lispworks-clim-6.1- n .i386.rpm

CLIM 2.0

Install this module if required by substituting the above filename into the same commands you used to install the LispWorks package (Installing or upgrading LispWorks for Linux).

If you used a --prefix argument when installing LispWorks, then use the same prefix for this module.
