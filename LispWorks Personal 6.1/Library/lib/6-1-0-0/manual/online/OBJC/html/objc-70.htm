trace-invoke

Function
Summary

Traces the invocation of an Objective-C method.

Package

objc

Signature

trace-invoke method

Arguments

method

A string.

Description

The function trace-invoke sets up a trace on invoke for calls to the Objective-C method named method . Use untrace-invoke to remove any such tracing.

See also

invoke
untrace-invoke
