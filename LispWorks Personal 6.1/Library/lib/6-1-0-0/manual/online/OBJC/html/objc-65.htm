retain

Function
Summary

Invokes the Objective-C "retain" method.

Package

objc

Signature

retain pointer => pointer

Arguments

pointer

A pointer to an Objective-C foreign object.

Values

pointer

An argument pointer .

Description

The function retain calls the Objective-C "retain" instance method of pointer to decrement its retain count. The pointer is returned.
