untrace-invoke

Function
Summary

Removes traces of the invocation of an Objective-C method.

Package

objc

Signature

untrace-invoke method

Arguments

method

A string.

Description

The function untrace-invoke removes any tracing on invoke for calls to the Objective-C method named method .
