image-freed-p

Function
Summary

Determines whether an image has been freed.

Package

graphics-ports

Signature

image-freed-p image => bool

Arguments

image

An image object.

Values

bool

A boolean.

Description

The image-freed-p function returns non- nil if the image has been freed, and nil otherwise.
