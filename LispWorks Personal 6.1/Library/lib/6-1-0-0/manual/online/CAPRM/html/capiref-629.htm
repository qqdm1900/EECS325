separation

Function
Summary

Returns the distance between two points.

Package

graphics-ports

Signature

separation x1 y1 x2 y2 => dist

Arguments

x1

An integer.

y1

An integer.

x2

An integer.

y2

An integer.

Values

dist

A real number.

Description

The separation function returns the distance between points ( x1 y1 ) and ( x2 y2 ).
