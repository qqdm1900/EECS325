apply-in-pane-process-if-alive

Function
Summary

Applies a function in the process associated with a pane.

Package

capi

Signature

apply-in-pane-process-if-alive pane function &rest args => nil

Description

The function apply-in-pane-process-if-alive applies function to args in the process that is associated with pane , if pane is alive.

This is like apply-in-pane-process except that function is called only if the pane is alive in the sense defined for the interface in execute-with-interface-if-alive. If pane does not have a process, then function is not called.
