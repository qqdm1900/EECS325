free-image

Function
Summary

Frees the library resources allocated with an image.

Package

graphics-ports

Signature

free-image port image

Arguments

port

A CAPI pane.

image

An image.

Description

The free-image function frees the library resources associated with image . This should be done when an image is no longer needed.
