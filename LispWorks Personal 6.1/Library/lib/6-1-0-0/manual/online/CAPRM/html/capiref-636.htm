transform-distances

Function
Summary

Transforms a list of alternating distance vectors by a given transform.

Package

graphics-ports

Signature

transform-distances transform distances => result

Arguments

transform

A transform.

distances

A list of pairs of real numbers.

Values

result

A list of pairs of real numbers.

Description

The transform-distances function transforms a list of alternating ( dx dy ) pairs in distances by the transform . Transformed values are returned as a new list.
