multi-line-text-input-pane

Class
Summary

A pane allowing several lines of text to be entered.

Package

capi

Superclasses

text-input-pane

Description

The multi-line-text-input-pane class behaves like a text-input-pane, except that the text entered by the user is allowed to span several lines -- that is, it is allowed to contain Newline characters.
