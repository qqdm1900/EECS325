current-document

Generic Function
Summary

Returns the current document of a MDI interface.

Package

capi

Signature

current-document mdi-interface => child

Arguments

mdi-interface

An instance of a subclass of document-frame.

Values

child

The current document of mdi-interface .

Description

The generic function current-document returns the top child interface of a MDI interface.

See also

document-frame
