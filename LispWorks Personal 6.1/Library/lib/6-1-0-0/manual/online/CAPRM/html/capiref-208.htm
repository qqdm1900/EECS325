item-pinboard-object

Class
Summary

An item-pinboard-object is a pinboard-object that displays a single piece of text.

Package

capi

Superclasses

pinboard-object
item

Description

The item-pinboard-object displays an item on a pinboard layout. It displays the text specified by the item in the usual way (either by the text field, or through printing the data with the print function).

Example
(capi:contain (make-instance
                'capi:item-pinboard-object
                :text "Hello World"))
(capi:contain (make-instance 'capi:item-pinboard-object
                             :data :red
                             :print-function 
                               'string-capitalize))
