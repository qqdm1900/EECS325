*editor-pane-default-composition-face*

Variable
Summary

The default composition face for editor-pane.

Initial Value

nil

Description

The variable *editor-pane-default-composition-face* gives the default composition face for all editor-pane s where the composition-face is set to :default .

:default is the default value for composition-face , so normally setting this variable affects the composition-face of all editor-pane s.

See editor-pane-default-composition-callback for a description of how it is used.
