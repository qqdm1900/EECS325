*update-screen-interfaces-hooks*

Variable
Summary

A list of functions that are called when a CAPI interface is created or destroyed.

Package

capi

Description

Each function in the list *update-screen-interfaces-hooks* is called when an interface interface is created or destroyed.

Each function takes two arguments: the screen and interface .

You should not remove system functions from this variable so take care if setting its value. Only add or delete your own functions.
