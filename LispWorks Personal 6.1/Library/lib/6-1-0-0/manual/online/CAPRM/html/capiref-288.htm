pane-string

Generic Function
Summary

Returns the text displayed in an editor-pane.

Package

capi

Signature

pane-string pane => text

Arguments

pane

An editor-pane.

Values

text

A string.

Description

The generic function pane-string returns as a string the text of the buffer that is currently displayed in the editor-pane pane .
