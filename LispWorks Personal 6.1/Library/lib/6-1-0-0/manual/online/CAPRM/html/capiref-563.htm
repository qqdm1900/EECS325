get-font-ascent

Function
Summary

Returns the ascent of a font.

Package

graphics-ports

Signature

get-font-ascent port &optional font => ascent

Arguments

port

A CAPI pane.

font

A font.

Values

ascent

An integer.

Description

The get-font-ascent function returns the ascent in pixels of the font associated with port , or the font given.
