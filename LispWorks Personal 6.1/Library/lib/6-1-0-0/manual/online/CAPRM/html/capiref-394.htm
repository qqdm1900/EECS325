set-pane-focus

Generic Function
Summary

Sets the input focus to a pane.

Package

capi

Signature

set-pane-focus pane

Arguments

pane

An instance of a subclass of simple-pane or choice.

Description

The function set-pane-focus sets the input focus to pane or one of its children.

See also
