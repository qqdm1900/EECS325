attach-interface-for-callback

Function
Summary

Changes the interface that is passed when a callback is made.

Package

capi

Signature

attach-interface-for-callback element interface

Description

The function attach-interface-for-callback changes the interface that is passed when a callback is made. Callbacks for element get passed interface instead of element 's parent interface.
