lower-interface

Function
Summary

The lower-interface function pushes the window containing a specified pane to the back of the screen.

Package

capi

Signature

lower-interface pane

Description

This pushes the window containing pane to the back of the screen. To bring it back use raise-interface, and to iconify it use hide-interface.
