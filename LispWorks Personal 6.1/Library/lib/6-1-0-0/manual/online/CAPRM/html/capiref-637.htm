transform-is-rotated

Function
Summary

Returns t if a given transform contains a rotation.

Package

graphics-ports

Signature

transform-is-rotated transform => bool

Arguments

transform

A transform.

Values

bool

A boolean.

Description

The transform-is-rotated function returns t if transform contains any rotation.
