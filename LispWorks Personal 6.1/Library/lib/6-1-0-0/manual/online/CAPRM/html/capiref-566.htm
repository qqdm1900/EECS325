get-font-height

Function
Summary

Returns the height of a font.

Package

graphics-ports

Signature

get-font-height port &optional font => height

Arguments

port

A CAPI pane.

font

A font.

Values

height

An integer.

Description

The get-font-height function returns the height in pixels of the font associated with port , or the font given.
