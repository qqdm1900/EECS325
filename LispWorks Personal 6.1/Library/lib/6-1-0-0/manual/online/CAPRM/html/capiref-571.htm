get-transform-scale

Function
Summary

Returns the overall scaling factor of a transform.

Package

graphics-ports

Signature

get-transform-scale transform => result

Arguments

transform

A transform object.

Values

result

A real number.

Description

The get-transform-scale function returns a single number representing the overall scaling factor present in the transform .
