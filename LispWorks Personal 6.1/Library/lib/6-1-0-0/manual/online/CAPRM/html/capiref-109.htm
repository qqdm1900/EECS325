*echo-area-cursor-inactive-style*

Variable
Summary

The drawing style of the Echo Area cursor when the window is inactive.

Package

capi

Initial Value

:invisible

Description

The drawing style of the cursor in the Echo Area of an inactive window in the LispWorks IDE.

The allowed values are :inverse , :outline , :underline and :invisible .
