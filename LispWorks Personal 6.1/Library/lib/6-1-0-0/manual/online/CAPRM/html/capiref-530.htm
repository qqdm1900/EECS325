draw-points

Function
Summary

Draws pixels or unit squares at given points.

Package

graphics-ports

Signature

draw-points port description &rest args

Arguments

port

A graphics port.

description

A description sequence.

args

graphics-state parameters passed as keyword arguments.

Description

The draw-points function draws several points (as if by draw-point) as specified by the description argument, which is a sequence of x y pairs. It is usually faster than several calls to draw-point. See draw-point for more information.
