unit-transform-p

Function
Summary

Returns t if a given transform is a unit transform.

Package

graphics-ports

Signature

unit-transform-p transform => bool

Arguments

transform

A transform.

Values

bool

A boolean.

Description

The unit-transform-p returns t if transform is the unit transform.
