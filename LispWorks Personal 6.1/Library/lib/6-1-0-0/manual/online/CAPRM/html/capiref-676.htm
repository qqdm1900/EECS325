color-model

Function
Summary

Returns the color-model for a color-spec.

Package

color

Signature

color-model color-spec => color-model

Arguments

color-spec

A color specification.

Values

color-model

:gray , :rgb , or :hsv .

Example
COLOR 29 > (color:make-gray 0.66667s0)
#(:GRAY 0.66667S0)
COLOR 30 > (color-model *)
:GRAY
COLOR 31 >
