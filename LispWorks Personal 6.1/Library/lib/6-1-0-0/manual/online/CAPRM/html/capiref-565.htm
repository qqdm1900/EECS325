get-font-descent

Function
Summary

Returns the descent in pixels of a font.

Package

graphics-ports

Signature

get-font-descent port &optional font => descent

Arguments

port

A CAPI pane.

font

A font.

Values

descent

An integer.

Description

The get-font-descent function returns the descent in pixels of the font associated with port , or the font given.
