pinboard-object-at-position

Generic Function
Summary

The generic function pinboard-object-at-position returns the uppermost pinboard object containing a specified point.

Package

capi

Signature

pinboard-object-at-position pinboard x y

Description

This function returns the uppermost pinboard object in the pinboard that contains the point specified by x and y . It determines this by mapping over every pinboard object within the pinboard until it finds one for which the generic function over-pinboard-object-p returns t .

Example
(setq pinboard 
      (capi:contain 
       (make-instance 
        'capi:pinboard-layout)
       :best-width 300
       :best-height 300))
(make-instance 'capi:item-pinboard-object
               :text "Hello world"
               :x 100 :y 100
               :parent pinboard)
(capi:pinboard-object-at-position pinboard 0 0) 
(capi:pinboard-object-at-position pinboard 110 110)
