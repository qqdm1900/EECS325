register-image-load-function

Function
Summary

Registers one or more image identifiers with an image loading function.

Package

graphics-ports

Signature

register-image-load-function image-id image-load-function
 &key image-translation-table

Arguments

image-id

An image identifier or a list of image identifiers.

image-load-function

A function.

image-translation-table

An image translation table.

Description

The register-image-load-function function registers one or more image-ids with an image-load-function in the image-translation-table . If image-load-function is nil it causes the default loader to be used in subsequent calls to load-image. The image-id argument can be a list of identifiers or a single identifier. The default value of image-translation-table is *default-image-translation-table*.
