graph-pane-edges

Function
Summary

Returns the edges of a graph.

Package

capi

Signature

graph-pane-edges graph-pane => edges

Arguments

graph-pane

A graph-pane .

Values

edges

A list.

Description

The function graph-pane-edges returns a list of all the graph-edge objects in the graph graph-pane.
