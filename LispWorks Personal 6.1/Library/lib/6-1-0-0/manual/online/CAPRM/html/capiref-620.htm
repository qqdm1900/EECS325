rectangle-left

Macro
Summary

Gets and set the left element of a rectangle.

Package

graphics-ports

Signature

rectangle-left rectangle => left

Signature

(setf rectangle-left) left rectangle => left

Arguments

rectangle

A rectangle.

Values

left

A real number.

Description

The rectangle-left macro returns and via setf sets the left element of rectangle .

rectangle is a list of numbers ( left top right bottom ).
