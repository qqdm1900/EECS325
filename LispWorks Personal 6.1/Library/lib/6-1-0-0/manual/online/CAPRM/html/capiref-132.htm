ensure-interface-screen

Function
Summary

Ensures that a top level interface is displayed on a given screen.

Package

capi

Signature

ensure-interface-screen self &key screen

Description

The function ensure-interface-screen ensures that the top level interface is displayed on the given screen (or the default) if display is called later without a screen argument. This allows the querying of font and color information associated with a particular screen. It returns the screen that is used.
