get-printer-metrics

Function
Summary

Returns the metrics for a printer.

Package

capi

Signature

get-printer-metrics printer

Description

The get-printer-metrics functions takes a printer as its argument and returns a printer-metrics object.

The metrics values in this object should be accessed by the printer-metrics readers.
