gesture-spec-p

Function
Summary

The predicate for Gesture Spec objects.

Package

system

Signature

gesture-spec-p object => result

Arguments

object

A Lisp object

Values

result

A boolean.

Description

The function gesture-spec-p is the predicate for whether the object object is a Gesture Spec object.
