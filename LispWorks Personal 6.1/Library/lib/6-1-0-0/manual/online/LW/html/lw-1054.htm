*main-process*

Variable
Summary

The process associated with the main thread.

Package

mp

Description

This special variable contains the process associated with the main thread of the application. On Mac OS X with the Cocoa GUI, this is the thread that runs the Cocoa event loop. On other platforms, this variable is always nil .
