barrier-name

Function
Summary

Returns the name of the barrier

Package

mp

Signature

barrier-name barrier => name

Arguments

barrier

A barrier.

Values

name

A string.

Description

The function barrier-name returns the name of the barrier, as supplied or defaulted in the call to make-barrier.

See also

barrier-wait
make-barrier
