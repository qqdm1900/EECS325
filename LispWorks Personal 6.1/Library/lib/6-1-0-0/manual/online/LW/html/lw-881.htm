appendf

Macro
Summary

Appends lists to the end of a given list.

Package

lispworks

Signature

appendf place &rest lists => result

Arguments

place

A place.

lists

A set of lists.

Values

result

An object.

Description

The modify macro appendf appends the lists given by lists to the end of the list in place . See append for more details.

See also

removef
