drop-table

Function
Summary

Deletes a table from a database.

Package

sql

Signature

drop-table table &key database =>

Arguments

table

The name of a table.

database

A database.

Values

None.

Description

The function drop-table deletes table from a database .

The default value of database is *default-database*.

See also

create-table
