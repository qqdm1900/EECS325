gesture-spec-modifiers

Function
Summary

Returns the modifiers in a Gesture Spec object.

Package

system

Signature

gesture-spec-modifiers gspec => mods

Arguments

gspec

A Gesture Spec object

Values

mods

An integer.

Description

The function gesture-spec-modifiers returns an integer representing the modifiers in the Gesture Spec object gspec .

The value mods contains some (or none) of the constants gesture-spec-accelerator-bit, gesture-spec-control-bit, gesture-spec-meta-bit, gesture-spec-hyper-bit, gesture-spec-shift-bit and gesture-spec-super-bit, combined as if by logior .
