unbreak-new-instances-on-access

Function
Summary

Removes the trapping installed by break-new-instances-on-access . Note that this function is deprecated.

Package

clos

Signature

unbreak-new-instances-on-access class-designator => t

Arguments

class-designator

The class whose trap you want to remove.

Values

Returns t .

Description

Removes the trapping installed by break-new-instances-on-access . Note that this function is deprecated. You should now use untrace-new-instances-on-access instead.
