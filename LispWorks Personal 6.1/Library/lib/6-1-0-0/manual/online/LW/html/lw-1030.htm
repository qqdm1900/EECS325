get-current-process

Function
Summary

Returns the current Lisp process.

Package

mp

Signature

get-current-process => result

Values

result

A mp:process , or nil .

Description

The function get-current-process returns the actual process in which it is called. In this respect it differs from *current-process*, which can be bound to another process. In

particular, when a process A calls the wait-function of process B, in the wait-function get-current-process returns the process A, but *current-process* is bound to process B.

result is nil if multiprocessing is off.

*current-process*
