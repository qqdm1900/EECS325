process-mailbox

Function
Summary

Accesses the mailbox associated with a process.

Package

mp

Signature

process-mailbox process => result

(setf process-mailbox) result process => result

Arguments

process

A process.

Values

result

A mailbox object, or nil .

Description

process-mailbox is an accessor function which returns or sets the mailbox associated with process .

Example
(setf (mp:process-mailbox mp:*current-process*)
      (mp:make-mailbox))
