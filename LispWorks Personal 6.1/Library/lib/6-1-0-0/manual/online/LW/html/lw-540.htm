23.12.2 Building 64-bit universal binary applications

To create a universal binary application from LispWorks (64-bit) for Macintosh, you will need to install 64-bit LispWorks on both a PowerPC Macintosh and an Intel-based Macintosh computer, and deliver on both machines. Contact Lisp Support if you need advice on creating a 64-bit universal binary after that.
