coerce

Function
Summary

Extends the standard coerce function, allowing it to take any Common Lisp type specifier.

Package

common-lisp

Signature

coerce object result-type => result

Arguments

object

A Lisp object.

result-type

A type specifier.

Values

result

An object of type result-type

Description

The coerce function still performs those conversions required by the standard, but a larger set of type specifiers is allowed for coercion. A type-error is signalled if result cannot be returned as the result-type specifies.
