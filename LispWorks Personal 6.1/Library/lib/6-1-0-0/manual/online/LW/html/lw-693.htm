*print-handler-frames*

Variable
Summary

Controls whether handler frames are printed in debugger output.

Package

dbg

Initial Value

nil

Description

This variable is used by the debugger when it displays the stack frames. Handler frames are created by error handlers (see The stack in the debugger), and are normally not shown by the debugger. However if *print-handler-frames* is set to t then the handler frames are displayed.

Example
USER 162 > (setq lw:*print-handler-frames* t) 
T 
USER 163 > (defun test (n)
   (handler-case (fn-to-use n)
(type-error () (format t "~%Type error~%") 0)))
TEST
