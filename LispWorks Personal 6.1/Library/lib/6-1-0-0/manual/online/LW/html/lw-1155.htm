38 The SQL Package

This chapter describes the symbols available in the SQL package which implements Common SQL. You should use this chapter in conjunction with Common SQL. In particular that chapter contains more information about the Oracle LOB interface (that is, those functions with names beginning sql:ora-lob- ).

On Microsoft Windows, Linux, x86/x64 Solaris, FreeBSD and Mac OS X, Common SQL is included only in LispWorks Enterprise Edition.
