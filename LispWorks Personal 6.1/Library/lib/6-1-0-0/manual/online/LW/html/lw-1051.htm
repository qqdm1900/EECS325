mailbox-reader-process

Function
Summary

Returns the reader process of a mailbox.

Package

mp

Signature

mailbox-reader-process mailbox => process

Arguments

mailbox

A mailbox.

Values

process

A process or nil .

Description

The function mailbox-reader-process returns the reader process of mailbox .
