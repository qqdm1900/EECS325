lock-name

Function
Summary

Returns the name of a lock.

Package

mp

Signature

lock-name lock => name

Arguments

lock

A lock object

Values

name

A string

Description

The function lock-name takes a lock object as its argument and returns the name of the lock object.

Example
(let ((lock (mp:make-lock :name "my lock")))
  (mp:lock-name lock))
=> "my lock"
