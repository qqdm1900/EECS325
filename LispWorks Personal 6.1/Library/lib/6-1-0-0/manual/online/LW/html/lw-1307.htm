stream-force-output

Generic Function
Summary

Implements force-output .

Package

stream

Signature

stream-force-output stream => nil

Arguments

stream

A stream.

Values

nil

Description

The generic function stream-force-output implements force-output . The default method is on fundamental-output-stream and does nothing.

There is an example in Stream output.
