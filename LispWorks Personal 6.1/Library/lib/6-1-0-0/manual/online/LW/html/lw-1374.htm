gesture-spec-hyper-bit

Constant
Summary

Used in the representation of a keystroke with the Hyper key.

Package

system

Description

The constant gesture-spec-hyper-bit is used to represent the Hyper modifier key in a Gesture Spec object.

See the entry for capi:output-pane in the CAPI Reference Manual for more information about the use of Gesture Specs.
