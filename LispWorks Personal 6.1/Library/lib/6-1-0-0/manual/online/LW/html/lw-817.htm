*profiler-threshold*

Variable
Summary

Controls which symbols are profiled on repeated profiling runs.

Package

hcl

Description

*profiler-threshold* is used with repeated profiling runs, to control which symbols are profiled. It is set by set-profiler-threshold.

See also

set-profiler-threshold
