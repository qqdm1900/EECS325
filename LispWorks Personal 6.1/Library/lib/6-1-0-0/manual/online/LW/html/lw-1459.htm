*multibyte-code-page-ef*

Variable
Summary

Holds the external format corresponding to the current Windows multi-byte code page.

Package

win32

Description

This variable holds the external format corresponding to the current Windows multi-byte code page. It is automatically initialized to the right value, when the image is started. If you change the code page (using _setmbcp ), you need to set this variable, too.
