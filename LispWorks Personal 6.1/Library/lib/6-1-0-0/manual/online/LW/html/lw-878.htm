8-bit-string

Type
Summary

The 8 bit string type.

Package

lispworks

Signature

8-bit-string length

Arguments

length

The length of the string (or * , meaning any).

Description

The type of strings that can hold simple chars of codes 0...255. This is the string type that is guaranteed to always take 8 bits per element.
