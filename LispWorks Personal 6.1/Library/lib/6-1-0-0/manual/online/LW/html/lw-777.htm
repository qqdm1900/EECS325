*default-profiler-sort*

Variable
Summary

The default sorting style for the profiler.

Package

hcl

Initial Value

:profile

Description

The variable *default-profiler-limit* controls which column of the profiler's columnar report is used for sorting.

The value can be one of :profile , :call or :top .
