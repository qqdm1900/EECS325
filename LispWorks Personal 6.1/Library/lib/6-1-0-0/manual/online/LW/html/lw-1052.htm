mailbox-send

Function
Summary

Sends an object to a mailbox.

Package

mp

Signature

mailbox-send mailbox object =>

Arguments

mailbox

A mailbox

object

An object

Description

The function mailbox-send sends object to mailbox . The object is queued in the mailbox for retrieval by the reader.
