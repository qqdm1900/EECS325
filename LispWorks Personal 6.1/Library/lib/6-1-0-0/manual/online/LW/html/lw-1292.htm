fundamental-binary-stream

Class
Summary

A class for binary streams.

Package

stream

Superclasses

fundamental-stream

Subclasses

fundamental-binary-input-stream
fundamental-binary-output-stream

Description

The class fundamental-binary-stream is the superclass of the binary input and output stream classes. A method for stream-element-type should be provided for instantiable subclasses of this class.
