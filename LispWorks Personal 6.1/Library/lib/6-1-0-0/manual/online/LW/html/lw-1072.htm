process-continue

Function
Summary

Wakes up a process.

Package

mp

Signature

process-continue process => nil

Arguments

process

A mp:process object.

Description

The function process-continue is wakes up the process process , regardless of whether it is sleeping, stopped or waiting.

process-continue returns nil .
