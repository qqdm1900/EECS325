true

Function
Summary

Ignores its arguments and returns t .

Package

lispworks

Signature

true &rest ignore => t

Arguments

ignore

All arguments are ignored.

Values

t

Description

The function true ignores all its arguments and returns t . It is useful as a functional argument.
