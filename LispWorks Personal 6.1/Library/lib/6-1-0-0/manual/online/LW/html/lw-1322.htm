stream-unread-char

Generic Function
Summary

Undoes the last call to stream-read-char .

Package

stream

Signature

stream-unread-char stream character => nil

Arguments

stream

A stream.

character

A character.

Values

nil

Description

The generic function stream-unread-char undoes the last call to stream-read-char , as in unread-char . Every subclass of fundamental-character-input-stream must define a method for this function.
