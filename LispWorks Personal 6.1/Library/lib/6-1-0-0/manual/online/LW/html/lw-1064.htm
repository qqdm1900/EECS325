map-process-backtrace

Function
Summary

Produces a backtrace for a process

Package

mp

Signature

map-process-backtrace process function

Arguments

process

A process

function

A function taking one argument

Values

None.

Description

The function map-process-backtrace collects a backtrace for the process specified by process , and the function function is called on each line of the backtrace in turn.

Example
CL-USER 1 > (mp:map-process-backtrace mp:*current-process* 'print)
