delete-records

Function
Summary

Deletes rows from a database table.

Package

sql

Signature

delete-records &key from where database =>

Arguments

from

A database table.

where

A SQL conditional statement.

database

A database.

Values

None.

Description

The function delete-records deletes rows from a table specified by from in which the where condition is true. The argument database specifies a database from which the records are to be removed, and defaults to *default-database*.
