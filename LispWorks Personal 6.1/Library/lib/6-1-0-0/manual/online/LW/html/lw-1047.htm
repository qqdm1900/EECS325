mailbox-empty-p

Function
Summary

Tests whether a mailbox is empty.

Package

mp

Signature

mailbox-empty-p mailbox => bool

Arguments

mailbox

A mailbox

Values

bool

A boolean

Description

The function mailbox-empty-p returns t if the given mailbox is empty and nil otherwise.
