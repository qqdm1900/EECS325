*print-command*

Variable
Summary

A command used for some printing operations.

Package

lispworks

Initial Value

"print" on Windows.

"lpr" on UNIX/Linux/Mac OS X/FreeBSD systems.

Description

This variable is used as the command sent by LispWorks to the shell in hardcopy-system .

See also

hardcopy-system
