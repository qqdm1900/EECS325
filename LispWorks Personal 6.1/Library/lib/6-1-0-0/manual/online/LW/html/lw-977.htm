stchar

Function
Summary

The accessor for simple text strings.

Package

lispworks

Signature

stchar string index => value

Arguments

string

A simple-text-string .

index

An index.

Values

value

The character in string at index .

Description

This is the accessor for simple text strings. setf is allowed.
