ipv6-address-p

Function
Summary

The predicate for objects of type ipv6-address.

Package

comm

Signature

ipv6-address-p object => result

Arguments

object

A Lisp object.

Values

result

A boolean.

Description

The function ipv6-address-p is the predicate for whether its argument object is of type ipv6-address.
