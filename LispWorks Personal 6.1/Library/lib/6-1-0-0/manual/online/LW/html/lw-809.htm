*max-trace-indent*

Variable
Summary

The maximum level of indentation used in trace output.

Package

hcl

Initial value

50

Description

*max-trace-indent* is the maximum indentation that is used during output from tracing. Typically each successive invocation of tracing causes the output to be further indented, making it easier to see how the calls are nested. The value o f *max-trace-indent* should be an integer.

Example
USER 8 > (setq hcl:*max-trace-indent* 4)
4
USER 9 > (defun sum (n res) (if (= n 0)
                       res
                      (+ n (sum (1- n) res))))
SUM
USER 10 > (trace sum)
SUM
USER 11 > (sum 3 0)
 0 SUM > (3 0)
   1 SUM > (2 0)
     2 SUM > (1 0)
     3 SUM > (0 0)
     3 SUM < (0)
     2 SUM < (1)
   1 SUM < (3)
 0 SUM < (6)
 6
Notes

*max-trace-indent * is an extension to Common Lisp.
