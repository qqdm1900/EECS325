drop-index

Function
Summary

Deletes an index from a database.

Package

sql

Signature

drop-index index &key database =>

Arguments

index

The name of an index.

database

A database.

Values

None.

Description

The function drop-index deletes index from database .

The default value of database is *default-database*.

See also

create-index
drop-table
