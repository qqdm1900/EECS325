gesture-spec-to-character

Function
Summary

Returns the character corresponding to a Gesture Spec object.

Package

system

Signature

gesture-spec-to-character gspec => char

Arguments

gspec

A Gesture Spec object

Values

char

A Lisp character.

Description

The function gesture-spec-to-character returns the Lisp character object corresponding to the Gesture Spec object gspec .

Modifier bits in gspec are mapped to Lisp character bits attributes where possible. gesture-spec-accelerator-bit is ignored.
