*default-package-use-list*

Variable
Summary

List of packages that newly created packages use by default.

Package

hcl

Initial Value

("CL" "LW" "HCL")

Description

This variable is the default value of the :use keyword to defpackage, which specifies which existing packages the package being defined inherits from.
