array-weak-p

Function
Summary

The predicate for whether an object is a weak array.

Package

hcl

Signature

array-weak-p object => result

Arguments

object

A Lisp object.

Values

result

A boolean.

Description

The function array-weak-p returns t if its argument object is a weak array, and otherwise returns nil .

See also

make-array
set-array-weak
