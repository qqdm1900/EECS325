dde-disconnect

Function
Summary

Disconnects a conversation object.

Package

win32

Signature

dde-disconnect conversation => result

Arguments

conversation

A conversation object.

Values

result

A boolean.

Description

The function dde-disconnect disconnects the conversation object. The conversation may no longer be used. If the conversation disconnects successfully, t is returned.
