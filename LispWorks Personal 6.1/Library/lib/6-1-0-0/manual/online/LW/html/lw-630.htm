apropos-list

Function
Summary

Searches for interned symbols.

Package

common-lisp

Signature

apropos-list string &optional package external-only => symbols

Arguments

string

A string designator.

package

A package designator or nil .

external-only

A generalized boolean.

Values

symbols

A list of symbols.boolean

Description

The function apropos-list behaves as specified in ANSI Common Lisp. There is an additional optional argument external-only , which if true restricts the search to symbols which are external in the searched package or packages. The default value of external-only is nil .
