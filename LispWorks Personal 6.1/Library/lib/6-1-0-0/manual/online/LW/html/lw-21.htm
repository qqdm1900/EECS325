2.3 The listener prompt

The variable *prompt* controls the appearance of the listener prompt. See *prompt* if you want to alter this.

If the default prompt contains a colon followed by a second positive integer then you are no longer in the top-level loop, but have entered the REPL debugger, as described in The Debugger.
