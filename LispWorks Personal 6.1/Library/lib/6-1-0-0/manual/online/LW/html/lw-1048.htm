mailbox-not-empty-p

Function
Summary

Tests whether a mailbox has contents.

Package

mp

Signature

mailbox-not-empty-p mailbox => bool

Arguments

mailbox

A mailbox

Values

bool

A boolean

Description

The function mailbox-not-empty-p returns nil if the given mailbox is empty and t otherwise.
