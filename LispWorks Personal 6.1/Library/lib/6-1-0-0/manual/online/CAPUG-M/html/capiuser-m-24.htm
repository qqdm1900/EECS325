3.2.2.1 Window titles

Specify a title for a CAPI window by supplying the :title initarg for the interface , and access it with interface-title .

Further control over the title of your application windows can be acheived by using set-default-interface-prefix-suffix and/or specializing interface-extend-title as illustrated in Controlling the interface title.
