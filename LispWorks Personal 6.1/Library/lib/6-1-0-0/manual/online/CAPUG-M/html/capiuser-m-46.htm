3.8.3 Tooltips for toolbar buttons

You can implement tooltips for a toolbar-button exactly as for collections and so on as described in Tooltips for collections, elements and menu items. Supply help-key for the toolbar-button and a help-callback for the interface . For an example of this see examples/capi/elements/toolbar.lisp .

However, if your toolbar-button s are grouped in a toolbar-component it is simpler to supply the :tooltips initarg. tooltips should be a list containing a string giving the tooltip text of each button in the component. For an example of this see examples/capi/applications/simple-symbol-browser.lisp .
