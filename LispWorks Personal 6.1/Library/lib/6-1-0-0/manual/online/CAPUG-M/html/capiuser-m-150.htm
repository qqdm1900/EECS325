11.2.4 Prompting for files

To prompt for a file, use the function prompt-for-file :

(prompt-for-file
 "Enter a file:")

You can also specify a starting pathname:

(prompt-for-file
  "Enter a filename:"
  :pathname (sys:get-folder-path :documents))
