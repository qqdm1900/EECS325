1.4 CORBA examples

The LispWorks ORB includes two example applications to help you start writing client/server applications in CORBA. We study these applications later in this guide.

Hello World

A client/server implementation of the standard Hello World application.

Bank

A three-tier client/server implementation of a banking application.

This manual walks you through these examples, which are ready to build and run straight out of the box.
