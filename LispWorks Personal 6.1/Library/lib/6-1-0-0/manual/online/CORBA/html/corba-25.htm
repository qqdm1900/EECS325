3.2 Where to find the example code

The bank example code developed in this tutorial can be found in LispWorks installation folder, under

lib/6-1-0-0/examples/corba

This folder has several subfolders.

  • bank , which contains the IDL that defines the CORBA interface to the server (the file bank.idl )
  • bank/client , which contains the implementation of the client
  • bank/server , which contains the implementation of the server
