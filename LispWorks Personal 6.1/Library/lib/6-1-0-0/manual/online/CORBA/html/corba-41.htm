5.1 The client

In this section, we design and implement a CORBA client. Our client presents a graphical user interface to a bank object and its operations. We implement the user interface using CAPI.

Because the primary motivation for this tutorial is to illustrate the use of CORBA, we focus less on the design of the graphical interface, and more on the method for interacting with CORBA objects.
