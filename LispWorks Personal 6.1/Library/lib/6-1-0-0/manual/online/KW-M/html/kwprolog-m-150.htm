*print-verbose*

Variable
Summary

Controls how much information is printed for an object.

Description

Normally objects in KnowledgeWorks are printed out in a brief form similar to ordinary CLOS objects. If this variable is set to t then all the slots and slot values are shown in its printed representation. Note that circularities cannot be detected.

Initial Value

nil
