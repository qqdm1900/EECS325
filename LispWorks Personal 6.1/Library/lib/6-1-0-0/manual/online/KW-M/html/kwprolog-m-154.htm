recency

Conflict Resolution Tactic / Function
Summary

Implements the recency tactic.

Signature

recency instantiation1 instantiation2 => result

Arguments

instantiation1

An instantiation object.

instantiation2

An instantiation object.

Values

result

A boolean.

Description

The function returns truel if and only if instantiation1 is preferred to instantiation2 by the conflict resolution tactic recency , otherwise false. The function is intended to be used primarily by including it in the conflict resolution strategy for a context.

Examples
(defcontext my-context1 :strategy (recency))
(defcontext my-context2 :strategy (priority recency))
