reset

Function
Summary

Clears all objects from the object base.

Signature

reset

Description

Clears all KnowledgeWorks objects (both KnowledgeWorks CLOS objects and KnowledgeWorks structures) from the object base of the current inferencing state.

The list of KnowledgeWorks classes remains unaffected.

Examples
(reset)
