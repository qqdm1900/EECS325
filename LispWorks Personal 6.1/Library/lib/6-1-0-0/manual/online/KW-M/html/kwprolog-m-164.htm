test

Backward Chaining Goal
Signature

test lisp-form

Arguments

lisp-form

A single lisp form.

Description

Succeeds if and only if the lisp-form returns a non-nil value. Any currently bound variables may be used in the lisp form.

Examples
(test (> ?c 10))
(test (not (and  (eq ?a ?b) (member ?b ?c))))
