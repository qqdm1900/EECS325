5.2 The Editor

Figure 5.3 KnowledgeWorks Editor

The KnowledgeWorks Editor is created by choosing Window > KnowledgeWorks > Editor . It is the same as the LispWorks Editor tool. Please see the LispWorks IDE User Guide for more information on the editor tool and the LispWorks Editor User Guide for information on the various editing commands.
