-mea

Conflict Resolution Tactic / Function
Summary

Implements the -mea tactic.

Signature

-mea instantiation1 instantiation2 => result

Arguments

instantiation1

An instantiation object.

instantiation2

An instantiation object.

Values

result

A boolean.

Description

The function returns true if and only if instantiation1 is preferred to instantiation2 by the conflict resolution tactic -mea , otherwise false. The function is intended to be used primarily by including it in the conflict resolution strategy for a context.

Examples
(defcontext my-context1 :strategy (-mea))
(defcontext my-context2 :strategy (priority -mea))
